# Learning Ruby

This repository contain all notes, code and projects of my learning process in the ruby programming language based on the course recommended by [ruby documentation](https://www.ruby-lang.org/en/documentation), the [Odin Project](https://www.theodinproject.com).

Although the course presents the Ruby on Rails course, initially I do not pretend to be providing these codes and notes here in this specific repository.

## Note on the exercises

All the exercises I will be doing across this learning process is cloned from the official [ruby-exercises git repo](https://github.com/TheOdinProject/ruby-exercises?tab=readme-ov-file#cloning-and-installation) provided by The Oding Project. 

All instructions will be inside of [the old readme](./ruby-exercises/README.md), they ask to fork the repository but, since I want to keep everything in this repo: I will just be deleting the `.git` directory and doing all of the exercises here, in this very repository.

You can notice that the exercises repository is inside of the top directory. I will be copying the exactly exercises directory inside of each respective module directory and solving them there (I would make symlinks but Linux doesn't allow hard links for directories. If does it is beyond my current knowledge).
By example: `2 Basic Ruby/Basic Data Types/Exercises` will be the same as `ruby-exercises/ruby-basics/1_data_types`.

All of the exercises contain RSpec TDD (Test Driven Development) and each individual exercise is a function with the necessary parameters and instructions.
